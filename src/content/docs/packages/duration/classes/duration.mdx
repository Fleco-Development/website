Everything in this package is based around the <InlineCode content="Duration" /> class, where you can create a duration and add/subtract time or get the end date in different formats.

**Type: [Object](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object)**

	lang="typescript"
	code={`import { Duration } from '@fleco/duration';

`}
