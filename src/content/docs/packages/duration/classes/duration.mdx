**Type: Class**

Everything in this package is based around the `Duration` class, where you can create a duration and add/subtract time or get the end date in different formats.

## Constructor

```ts
new Duration('1w 2d');
//OR
new Duration({ weeks: 1, days: 2 });
```

| Parameter | Type                                                                                                                                                        | Optional |
| --------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- | -------- |
| duration  | [DurationOptions](/packages/duration/duration-options) \| [String](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) | **No**	 |
